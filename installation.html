Installation¶

Todo

Include a link to the pypi page in the very next sentence.

mannkendall (Python) will be available on pypi, which should make its installation straighforward. In a terminal, type:

pip install mannkendall

And that will take care of things. mannkendall (Python) uses semantic versioning. The latest stable version is 1.0.0.

The most recent release of mannkendall (Python) is also available for download from its Github repository.

Requirements¶

mannkendall (Python) is compatible with the following python versions:

    python_requires='>=3.8.0',

Furthermore, mannkendall (Python) relies on the following external modules, which will be automatically installed by pip if required:

    install_requires=["numpy>=1.19.2",
                      "scipy>=1.5.0",
                      "statsmodels>=0.12.0",
                      "pytest"],

Testing the installation¶

The most basic test you can do, to see if the installation was successful, is to verify the version of the package:

import mannkendall
mannkendall.__version__
